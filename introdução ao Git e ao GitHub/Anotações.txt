O Git Bash é um terminal extendido para otimizar o uso do Git.

Comandos básicos para um bom desempenho no GitBash.

Windows!

CD - Change directory (Mudar diretório)
DIR - Lista repositório
MKDIR - Criar diretório
DEL/RMDIR - Para deletar/ Remover diretórios

LINUX!

CD - Change directory (Mudar diretório)
LS - Lista repositório
MKDIR - Criar diretório
RM/RF - Para deletar/ Remover diretórios
